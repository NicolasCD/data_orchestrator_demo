# Data_orchestrator_demo

## Object

### Function_1
Create a S3 bucket named 'example-titanic'. 
Add file 'https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv' into bucket.

### Function_2
Pull file from S3 'example-titanic/titanic.csv'
Show stats.

## Prerequis

### Tools 
* Linux
* Docker
* Zip
* Curl
* Scw
* Scw-do
### Rights
* Serverless Functions read/write
* Data Orchestrator read/write
* Object storage read/write
